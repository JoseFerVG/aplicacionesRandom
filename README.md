## 🌟 Sobre el Proyecto

**Apps Random** es mi repositorio personal de proyectos frontend experimentales. Un lugar donde albergar ideas sueltas, herramientas de la vida cotidiana o simplemente pruebas de diseño y estética web. 

Todo el código está concebido bajo una **filosofía limpia**:
- ✨ **Sin Frameworks pesados**: Todo es HTML Semántico, CSS Puro y Vanilla JavaScript.
- 🍪 **Sin Cookies ni Rastreadores**: Cero *tracking* o venta de datos.
- 🎨 **Estilo Cuidado**: Diseños responsive, desde Neomorfismo hasta Brutalismo Pop-Art.

---

## 🚀 Aplicaciones Incluidas

Actualmente el ecosistema cuenta con los siguientes proyectos funcionales:

### 🍣 1. [Calculadora Buffet Sushi](./calculadoraChuchi)
¿Alguna vez te has preguntado si le sales rentable al restaurante cuando vas a un buffet libre? 
Esta calculadora realista estima el coste de la materia prima que consumes frente al precio del menú.
- 💸 **Base de Precios Real:** Estimaciones de coste basadas en mercado.
- 📊 **Cálculo de Margen:** Calcula la ganancia/pérdida del restaurante.
- 📱 **Interfaz Premium:** Diseño moderno y completamente adaptable a móviles.

### 🍱 2. [Sushímetro / Bunker Sushi Pop-Art](./contadorPiezas)
Un contador multi-piezas para ir registrando lo que comes en tiempo real mientras disfrutas de un buffet. Olvídate de perder la cuenta.
- 💥 **Estética Brutalista / Neo-Pop-Art:** Colores vibrantes, sombras duras, tipografías marcadas.
- ⚡ **Animaciones Hardware-Accelerated:** Sensación táctil (POW!) hiper-responsiva en cada pulsación.
- 📸 **Generador de Tickets:** Exporta tu comida a un ticket en formato imagen de alta resolución para presumir de lo mucho que has devorado.

## 🛠 Instalación y Uso

Dado que son aplicaciones estáticas Vanilla front-end, no necesitas instalar dependencias complicadas ni construir ningún bundle para usarlas.

1. Selecciona tu app y ábrela localmente sirviéndole un host estático:
   ```bash
   # Si tienes Node instalado (recomendable)
   npx http-server ./
   
   # Si prefieres Python
   python -m http.server 8000
   ```
2. O simplemente abre el archivo `index.html` en tu navegador para ver la portada.

> [!NOTE]
> Para la funcionalidad de guardar imagen en el **Sushímetro**, es necesario ejecutar las páginas bajo un servidor web (HTTP/HTTPS) en lugar de un protocolo de archivo local (`file:///`) para que la librería html2canvas no se bloquee por las políticas CORS de seguridad del navegador.
